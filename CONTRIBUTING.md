# Contributing to OpenModelica Simulation Studio

Thank you for your interest in contributing to this project!

## Contribution Guidelines

1. **Bug Reports**: If you find a bug, please create an issue with clear steps to reproduce it.
2. **Feature Requests**: We welcome ideas for new features. Please open an issue to discuss them first.
3. **Pull Requests**:
   - Ensure your code follows PEP 8 standards.
   - Include docstrings for new modules or functions.
   - Test your changes thoroughly before submitting.
   - Maintain the existing OOP architecture.

## Development Setup

1. Fork the repository.
2. Install development dependencies: `pip install -r requirements-dev.txt`.
3. Set up a virtual environment: `python -m venv .venv`.

We appreciate your effort in improving the OpenModelica Simulation Studio!
